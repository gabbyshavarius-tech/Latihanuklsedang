# Latihanuklsedang
Program Faktorial

Fungsi:
Untuk menghitung hasil faktorial dari bilangan bulat positif yang dimasukkan pengguna.

Cara kerja:
Program meminta input bilangan, lalu melakukan perulangan dari 1 hingga bilangan tersebut dan mengalikan setiap nilainya untuk mendapatkan hasil faktorial (n!).


  Program Volume Tabung

Fungsi:
Untuk menghitung volume tabung menggunakan fungsi dengan parameter jari-jari dan tinggi.

Cara kerja:
Program meminta input jari-jari dan tinggi, lalu memanggil fungsi hitungVolume() yang menghitung volume dengan rumus π × r² × t dan menampilkan hasilnya.
